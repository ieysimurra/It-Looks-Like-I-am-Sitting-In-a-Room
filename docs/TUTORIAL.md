### Using Real-time Mode

1. Record something first
2. Click **"üî¥ Real-time Mode"** or press `E`
3. Your recording plays in a continuous loop
4. Adjust any slider and hear changes instantly:
   - **Filter Q**: Higher = more resonant
   - **Dry/Wet**: Higher = more effect
   - **Room Presets**: Click to switch rooms
5. Press **Space** to exit

### Adjusting Effect Parameters

#### Filter Q (5-100)
- **Low (5-20)**: Subtle, natural coloring
- **Medium (30-50)**: Noticeable resonances
- **High (60-100)**: Dramatic, ringing effect

#### Dry/Wet (0-100%)
- **0%**: No effect (original sound)
- **50%**: Balanced mix
- **100%**: Fully processed (maximum effect)

#### Feedback (50-100%)
- Controls overall output level
- Higher values = louder, more aggressive transformation

#### Room Presets

| Preset | Best For |
|--------|----------|
| Small Room | Quick transformations, bright tones |
| Large Hall | Slow, deep transformations |
| Bathroom | Metallic, echoey character |
| Stairwell | Harmonic, musical results |
| Cathedral | Deep drones, meditative sounds |

### Using Loop Playback

1. Process several iterations first
2. Click **"üîÅ Play All Loop"**
3. Hear all iterations play in sequence, repeating
4. Click **"‚èπ Stop Loop"** or press **Space** to stop

### Exporting Your Work

| Button | Result |
|--------|--------|
| **Current WAV** | Download current iteration as WAV file |
| **All ZIP** | Download all iterations in a ZIP archive |
| **Sequence** | Single WAV with all iterations concatenated |
| **Image** | PNG screenshot of the visualization |

### Tips for Best Results

1. **Speak clearly** during recording
2. **Use higher Q values** (50-80) for more dramatic changes
3. **Set Dry/Wet to 80-100%** for noticeable effects
4. **Process 8-12 iterations** for full transformation
5. **Try Cathedral preset** for deep, drone-like results
6. **Use Real-time Mode** to find your ideal settings before processing
